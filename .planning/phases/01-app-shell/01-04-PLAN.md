  <what-built>
    Complete Tauri 2 app shell with system tray, positioned popup, close-to-tray lifecycle, ConfigStore persistence, and global hotkey stub. Plans 01-03 implemented:
    - Plan 01: Project scaffold + all plugins installed + capabilities configured
    - Plan 02: Rust tray backend (TrayIconBuilder, positioned popup, quit menu, close-to-tray intercept)
    - Plan 03: React frontend (TrayPopup UI, ConfigStore init, hotkey registration)
  </what-built>
  <how-to-verify>
Run `pnpm tauri dev` from the project root and verify each criterion:

**Criterion 1 — Tray icon visible, no window on startup (APP-01, APP-06)**
- Launch `pnpm tauri dev`
- Expected: No window appears. Speakly icon appears in menu bar (macOS) or system tray (Windows)
- PASS if: Tray icon visible within 5 seconds, zero windows on screen

**Criterion 2 — Clicking the tray icon opens the popup (APP-01)**
- Left-click the tray icon
- Expected: A small ~300x200px popup window appears positioned near the tray icon
- Popup shows: grey dot + "Bereit" text at top, empty preview area with placeholder text, gear icon at bottom-right
- PASS if: All three elements visible without scrolling

**Criterion 3 — Close hides to tray, does not quit (APP-06, D-05, D-07)**
- With popup open, click the close button (red dot on macOS, X on Windows)
- Expected: Popup closes, tray icon STILL VISIBLE, process still running
- FAIL if: Tray icon disappears (process exited)
- PASS if: Tray icon still visible after popup close; clicking tray again reopens popup

**Criterion 4 — Quit via tray right-click menu (D-06)**
- Right-click the tray icon
- Expected: Context menu appears with "Speakly beenden" item
- Click "Speakly beenden"
- Expected: App closes completely — tray icon disappears
- PASS if: App exits cleanly after menu click

**Criterion 5 — ConfigStore persists test key across restarts**
- After the first launch, check the settings file exists with correct content
- On macOS: `cat ~/Library/Application\ Support/de.speakly.app/settings.json`
- Expected: File contains `"initialized":true` and `"hotkey":"CommandOrControl+Shift+Space"`
- Restart the app (`pnpm tauri dev` again), settings file should still exist with same content
- PASS if: File present with expected keys before AND after restart
  </how-to-verify>
