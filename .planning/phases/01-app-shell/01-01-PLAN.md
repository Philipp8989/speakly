**Step 1: Ensure Rust is installed**
Run: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y`
Then: `source "$HOME/.cargo/env"`
Verify: `rustc --version` returns 1.77+ and `cargo --version` works.

**Step 2: Scaffold the Tauri project**
From /Users/philipppotzinger/Desktop/speakly/ run:
`pnpm create tauri-app@latest . --template react-ts --manager pnpm --yes`
This scaffolds into the current directory. If the CLI asks for app name use "speakly", identifier "de.speakly.app".

**Step 3: Install Tauri plugins (JS side)**
```
pnpm add @tauri-apps/plugin-store @tauri-apps/plugin-positioner @tauri-apps/plugin-global-shortcut
```

**Step 4: Add Rust plugin crates to src-tauri/Cargo.toml**
In the `[dependencies]` section add:
```toml
tauri-plugin-store = "2"
tauri-plugin-positioner = { version = "2", features = ["tray-icon"] }
tauri-plugin-global-shortcut = "2"
serde_json = "1"
```

**Step 5: Configure tauri.conf.json**
Set the main window config under `app.windows[0]` to exactly:
```json
{
  "label": "main",
  "title": "Speakly",
  "width": 300,
  "height": 200,
  "visible": false,
  "decorations": false,
  "resizable": false,
  "alwaysOnTop": true,
  "skipTaskbar": true
}
```
Set `app.trayIcon`:
```json
{
  "iconPath": "icons/icon.png",
  "iconAsTemplate": false,
  "menuOnLeftClick": false
}
```
Set `app.withGlobalTauri`: false (use ESM imports, not global).
Set `identifier`: "de.speakly.app"

**Step 6: Configure plugin permissions in src-tauri/capabilities/default.json**
The file must contain these permissions in the `permissions` array:
```json
"store:default",
"positioner:default",
"global-shortcut:allow-register",
"global-shortcut:allow-unregister",
"global-shortcut:allow-is-registered",
"core:default"
```
Full example structure:
```json
{
  "$schema": "../gen/schemas/desktop-schema.json",
  "identifier": "default",
  "description": "Standardberechtigungen fuer Speakly Desktop",
  "windows": ["main"],
  "permissions": [
    "core:default",
    "store:default",
    "positioner:default",
    "global-shortcut:allow-register",
    "global-shortcut:allow-unregister",
    "global-shortcut:allow-is-registered"
  ]
}
```

**Step 7: Replace src/App.tsx with a minimal stub**
```tsx
// Speakly App Root — Phase 1 Stub
// Vollstaendige UI kommt in Plan 03
function App() {
  return (
    <div className="flex items-center justify-center h-screen bg-gray-900 text-white">
      <p className="text-sm text-gray-400">Speakly wird geladen...</p>
    </div>
  );
}

export default App;
```

**Step 8: Verify the build compiles**
Run: `pnpm tauri dev --no-watch` and confirm it compiles without errors (Ctrl+C after first successful build).
If compilation fails due to missing icons, create placeholder: `cp src-tauri/icons/icon.png src-tauri/icons/icon.png` (icons directory is created by scaffold).

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Scaffold script → disk | pnpm create tauri-app writes files to project directory |
| Rust install script → system | curl | sh installs Rust globally; user must review before running |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | Rust install script (curl pipe to sh) | mitigate | Use official rustup.rs URL only; verify HTTPS; user should inspect before running on sensitive machines |
| T-01-02 | Tampering | src-tauri/capabilities/default.json | mitigate | Grant only required permissions (store:default, positioner:default, global-shortcut:allow-register); no wildcard permissions |
| T-01-03 | Information Disclosure | tauri-plugin-store plain JSON file | accept | Phase 1 only stores non-sensitive config (initialized: true, hotkey string); API keys are NOT stored in Phase 1; re-assess in Phase 2 |
| T-01-04 | Spoofing | Tray icon (fake app spoofing) | accept | Not mitigable at app level; macOS/Windows code signing (Phase 7) provides partial mitigation; user education |
</threat_model>
