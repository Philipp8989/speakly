```rust
use std::collections::HashMap;

pub fn get_system_prompt(command_id: &str) -> Result<String, String> {
    let mut map = HashMap::new();
    map.insert("formal",           "Formuliere den folgenden Text professioneller und formeller um. Behalte den Inhalt bei, aber verwende eine gehobene, sachliche Sprache. Antworte nur mit dem umformulierten Text, ohne Erlaeuterungen.");
    map.insert("kuerzer",          "Kuerze den folgenden Text auf das Wesentliche. Behalte alle wichtigen Informationen, entferne aber Wiederholungen und unnoetige Details. Antworte nur mit dem gekuerzten Text.");
    map.insert("translate_en_de",  "Uebersetze den folgenden Text ins Deutsche. Achte auf natuerliche, idiomatische Sprache. Antworte nur mit der Uebersetzung.");
    map.insert("translate_de_en",  "Translate the following text into English. Use natural, idiomatic language. Respond only with the translation.");
    map.insert("email",            "Forme den folgenden Text in eine professionelle E-Mail um. Fuege eine passende Anrede und einen Abschluss hinzu. Antworte nur mit dem E-Mail-Text.");
    map.insert("slack",            "Forme den folgenden Text in eine lockere, praegnante Slack-Nachricht um. Nutze informelle Sprache, halte sie kurz. Antworte nur mit dem Slack-Text.");
    map.get(command_id)
        .map(|s| s.to_string())
        .ok_or_else(|| format!("Unbekannter AI-Command: {}", command_id))
}
```

**2. Public async function `call_claude_api` (per D-08, D-09, D-10, D-11, D-12):**
```rust
pub async fn call_claude_api(
    app: &tauri::AppHandle,
    command_id: &str,
    text: &str,
) -> Result<String, String>
```

- Read `anthropic_api_key` from ConfigStore `settings.json` (same pattern as stt/mod.rs line 13-20)
- Return `Err("Anthropic API-Key fehlt oder ungueltig".into())` if empty
- Get system prompt via `get_system_prompt(command_id)?`
- Build JSON body:
```json
{
  "model": "claude-3-5-haiku-latest",
  "max_tokens": 4096,
  "system": "<system_prompt>",
  "messages": [{"role": "user", "content": "<text>"}]
}
```
- POST to `https://api.anthropic.com/v1/messages` with:
  - Header `x-api-key: <key>`
  - Header `anthropic-version: 2023-06-01`
  - Header `content-type: application/json`
  - Timeout: 15 seconds (per D-12)
- On timeout/network error: `Err("Netzwerkfehler — AI-Umformulierung fehlgeschlagen".into())`
- On HTTP 401/403: `Err("Anthropic API-Key fehlt oder ungueltig".into())`
- On other HTTP error: `Err("Netzwerkfehler — AI-Umformulierung fehlgeschlagen".into())`
- Parse response: `body["content"][0]["text"].as_str()` → `Ok(text.to_string())`
- If parse fails: `Err("Ungueltiger AI-Response".into())`

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-03 | DoS | Claude API timeout | mitigate | 15-second timeout (D-12) prevents indefinite blocking; on timeout paste raw transcript |
| T-05-04 | Elevation of Privilege | text payload | accept | text is user's own transcript — no server-side persistence, no PII handling beyond user's own words |
