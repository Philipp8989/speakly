<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| build artifacts → installer | Unsigned binary in .dmg — Gatekeeper blocks until notarized |
| CI runner → Apple notarization service | notarytool submits binary to Apple over HTTPS |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-01 | Tampering | .dmg artifact | mitigate | notarization staples Apple ticket — any post-build modification breaks notarization |
| T-07-02 | Repudiation | build provenance | accept | GitHub Actions log provides build audit trail; acceptable for internal tool |
| T-07-03 | Info Disclosure | Cargo.toml / tauri.conf.json | accept | no secrets in these files; all signing secrets stay in CI env vars |
</threat_model>
