<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete Phase 3 recording pipeline:
- Rust recording module (cpal audio capture, RMS VAD, hound WAV encoding, silence auto-stop)
- Hotkey handler in Rust: toggle mode (press/press) and hold mode (hold/release)
- TrayPopup with animated audio level bars and recording state indicators
- JS hotkey stub removed; Rust owns hotkey entirely
  </what-built>
  <how-to-verify>
Test each of the 5 Phase 3 success criteria in order. Run `cargo tauri dev` if not already running.

**Criterion 1 — Toggle Mode (REC-01):**
1. Press CommandOrControl+Shift+Space once
2. Observe: tray popup status changes to "Aufnahme..." with red pulsing dot
3. Press the hotkey again
4. Observe: status returns to "Bereit" (gray dot)
PASS if: first press starts, second press stops.

**Criterion 2 — Hold-to-Record / Push-to-Talk (REC-02):**
1. Press and HOLD CommandOrControl+Shift+Space for ~2 seconds while speaking
2. Observe: recording starts immediately, audio bars animate
3. RELEASE the hotkey
4. Observe: recording stops immediately on release
PASS if: hold starts, release stops (not another press).

**Criterion 3 — Tray Icon State Changes (REC-03 visual + D-14):**
1. Start a recording (toggle or hold)
2. Observe the system tray area
3. Stop the recording
PASS if: tray icon visually differs between idle and recording states
(Note: processing state may show briefly or not — Phase 4 wires STT pipeline)

**Criterion 4 — Audio Level Bars (REC-03 waveform):**
1. Start a recording and speak normally
2. Observe the TrayPopup
PASS if: animated vertical bars appear and their height responds visibly to speech volume.
Tap the desk or snap fingers near mic to create sharp audio peaks — bars should jump.

**Criterion 5 — Silence Auto-Stop and VAD Discard (REC-05 + D-10, D-11):**
Test A — Silence auto-stop:
1. Start a recording, say nothing
2. Wait ~3 seconds
3. Observe: recording stops automatically
PASS if: recording stops without pressing hotkey again.

Test B — Discarded toast:
1. Start a recording, immediately stop it (< 0.5 seconds after starting)
2. Observe the popup
PASS if: brief "Zu kurz" message appears in the popup.
